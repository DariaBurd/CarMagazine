# CARMAGAZINE

## Учебное консольное приложение

### [!] ИНФОРМАЦИЯ О ПРОЕКТЕ

**CarMagazine** - это учебное консольное приложение для работы с базой данных автомобилей. Разработано для изучения Java, ООП, многопоточности и работы с коллекциями.

- **Лицензия**: MIT
- **GitHub**: [https://github.com/DariaBurd/CarMagazine/](https://github.com/DariaBurd/CarMagazine/)

---

### [!] ВОЗМОЖНОСТИ ПРИЛОЖЕНИЯ

- **ЗАГРУЗКА ДАННЫХ**:
  - Генерация случайных автомобилей (мощность, год, цвет, комплектация)
  - Ручной ввод с валидацией (проверка корректности данных)
  - Загрузка из текстового файла (формат: мощность, год, комплектация, цвет)

- **ПРОСМОТР**:
  - Отображение всех автомобилей
