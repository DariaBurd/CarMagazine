Сортировка автомобилей с помощью разных способов с использованием паттерна Стратегия (Strategy).

Программа берёт список автомобилей и умеет сортировать его пятью разными способами:

По мощности (от меньшей к большей)
По году выпуска (от старых к новым)
По комплектации (по алфавиту)
По цвету (по алфавиту)
Дополнительно задание: только чётные годы сортируются по возрастанию, нечётные остаются на своих местах

Основные части кода:

SortStrategy — интерфейс для всех стратегий сортировки.
Отдельные классы для каждого способа:
• SortByPower
• SortByYear
• SortByConfiguration
• SortByColor
• AdditionSortByColor (доп. задание)

AutomobileSort — основной класс, который:
• запоминает, какой способ сортировки сейчас выбран
• позволяет менять способ через setStrategy()
• делает копию списка перед сортировкой
• выводит отсортированный результат
Test — демонстрация:
создаёт некоторое количество разных машин далее показывает исходный список потом применяет все 5 способов сортировки по очереди и выводит результаты.
