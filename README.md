1. Пакет catalog — создание и хранение автомобилей

CarFactory.java — фабрика для создания автомобилей тремя способами:

- Случайная генерация — автомобили со случайными параметрами (мощность 50-500 л.с., год 1886-2026, случайные комплектация и цвет)

- Ручной ввод — построчный ввод данных с консоли с валидацией каждого поля

- Загрузка из файла — чтение данных из текстового файла формата мощность,год,комплектация,цвет

CarCollection.java — кастомная коллекция для хранения автомобилей:

- Реализует AbstractCollection<Automobile>

- Позволяет сортировать коллекцию через стратегии (sortWith())

- Имеет методы фильтрации (filterByYear, filterByPower)

- Поддерживает Stream API (stream(), parallelStream())

InputValidator.java — валидация входных данных:

- Проверка мощности (число > 0)

- Проверка года (от 1886 до 2026)

- Проверка комплектации и цвета (не пустые)
