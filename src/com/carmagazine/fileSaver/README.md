# package com.carmagazine.fileSaver 
— сохранение данных в файл.

## Информация о пакете
**Класс FileSaver в пакете com.carmagazine.fileSaver** реализует дополнительное задание 2 — запись отсортированных коллекций и найденных значений в файл в режиме добавления данных (append).

## Основные возможности:
- **Метод saveSortedCars()** — сохраняет отсортированную коллекцию автомобилей с указанием даты, типа сортировки и количества записей.
- **Метод saveSearchResults()** — сохраняет результаты поиска с указанием даты, типа поиска, поискового запроса и количества записей.
- **Метод saveCurrentCars()** — сохраняет текущий список автомобилей (актуальную коллекцию).

## Особенности реализации:
- Использование try-with-resources для автоматического закрытия потоков.
- Режим append (FileWriter(fileName, true)) — данные добавляются в конец файла, не перезаписывая существующие.
- Валидация пути к файлу (проверка на пустоту и расширение .txt).
- Информативное форматирование: дата, заголовки, нумерация, разделители, общее количество записей.
- Корректная обработка пустых коллекций.
- Обработка ошибок ввода-вывода с понятными сообщениями.

Файлы с данными можно сохранить как в корневой директории проекта, так и в любой директории на диске, указав полный путь.
