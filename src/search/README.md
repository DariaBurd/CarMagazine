# package search 
— реализация паттерна Strategy (Стратегия) для поиска автомобилей.

## Информация о пакете
**Пакет search** предоставляет гибкую систему поиска автомобилей по различным критериям.

## Основные компоненты:
- **SearchStrategy** — интерфейс стратегии поиска, определяющий метод search() и getDescription().
- **SearchManager** — контекст, управляющий всеми стратегиями поиска (регистрация всех стратегий, меню, проверка выбора).
- **MainSearch** — модуль интеграции поиска в консольное меню.

## Доступные стратегии поиска:
- **SearchByPowerRange** — Поиск по мощности (точное совпадение, диапазон, больше/меньше)
- **SearchByYear** — Поиск по году выпуска (точное совпадение)
- **SearchByConfiguration** — Поиск по комплектации (частичное совпадение)
- **SearchByColor** — Поиск по цвету (частичное совпадение)

## Тестирование
**SearchTest** — ручные тесты для проверки всех стратегий поиска.
